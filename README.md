# RedShot
Try to avoid the multi-color enemies in this 3d-style game!
Link to try it out: https://red-shot-phi.vercel.app/

# How to play:
- use mouse up down to move forward and backward
- use mouse left and right to move side to side
- try to nto fall off the platform
- press space to jump over enemies

AND MOST OF ALL, DON'T FORGET TO HAVE FUN! 🤩
